Репозиторий содержит три независимых папки с заданиями:
- `task1-kafka-playground` — локальный Kafka стенд через Docker Compose: создание топика, отправка/чтение сообщений, анализ метаданных.
- `task2-go-microservice` — Go REST API с CRUD, rate limiting, метриками Prometheus и интеграцией MinIO.
- `task3-go-k8s-analytics` — Go сервис с потоковой аналитикой (rolling average + z-score), Redis-кэшем и манифестами Kubernetes (HPA).
